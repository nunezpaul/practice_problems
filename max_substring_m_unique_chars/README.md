Find the longest substring with k unique characters in a given string
Given a string you need to print the length of the longest possible substring that has exactly M unique characters.

  Examples:

      "aabbcc", k = 1
      answer = 2

      "aabbcc", k = 2
      answer = 4

      "aabbcc", k = 3
      answer = 6

      "aaabbb", k = 3
      answer = 6
